# hand_feature-system
手势识别系统


# Description


 本篇README.md面向开发者

### 上手指南

下载过后，需在终端进行多线程运作，启动过后出现本地网页！
代码内容可以自行制作，修改。

## 功能描述

-通过摄像头识别单手的动作

###### 开发前的配置要求

1. 需要摄像头(清晰)，保证基础的渲染即可。

###### **安装步骤**

1.获取Pycharm等等可编译py文件的软件编译器，
2.进行下载，并使用编译器打开

```sh
git clone https://github.com/hand_feature
```


### 部署

需要streamlit run your_script.py [-- script args]

### 使用到的框架

- python3.8以上
